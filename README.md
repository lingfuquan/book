# **疫情期间：一个运维做空瑞幸咖啡股价的过程**

\#  作者：Mr~Tiger

\#  字数八千多字，全程阅读约需9分钟

 

笔者先简单的自我做个介绍吧，笔者也是一名从事运维工程师岗位的“菜鸟”，跟github上面很多的“大神”还有段距离，然后业余爱好喜欢捣腾一下投资理财，还有粤菜的研究。因为不会玩游戏，也不喜欢游戏，所以业余就只能捣鼓一下这些玩意了，没想到经过时间的积累，还就探索到了自己的那一套“理论”了。今天有时间就分享一下吧。

大家看到的标题，自然也是在疫情期间，“IT男”玩了一下炒股的故事了。具体的起因是这样的，笔者在年前就已经离职了，想在年后跳槽的。结果刚回到老家没三天，这个疫情的新闻一下涌现到各大新闻媒体，互联网了，所以一个接着一个的“通知”就不断出来了。笔者也就一直在家待着，哪里都不去。所以就尝试把以前掌握的“投资理论”拿出来“实验”一下了。笔者就把那些“金融佬”常用的“借鸡生蛋”技巧来操作。

首先资金的准备。第一：笔者作为一个运维工程师，也算积累到一点点小钱，但是这点小钱也不敢冒然直接拿出来投资了，农村人的出身，多少也会有一些冒不起这个风险的。但笔者也是一个爱冒险的人，也喜欢尝试一下新鲜的玩意。说到这里，可能会有一点的矛盾体了，所以要解决矛盾就是要想想如何把风险降低一点。所以笔者首先是在各大平台（某宝，某白条，某卡）把一些额度给套现出来，总金额1万的美金。平常笔者有习惯用这些平台，所以也就积累了一些信用，额度也就涨上来了。第二：笔者这样做的目的，是为了减少现金流的风险，如果一股脑的把自己的全部小钱拿出来了，一旦失利，将会是灭顶之灾。所以为了平衡现金流，就不得不要找一些第三方的平台来借一下“母鸡”了。哪怕最坏的结果，母鸡在笔者手上死了之后，自己那点小钱还可以做个赔付。所以稳定的现金流是极其的重要了，也可以把风险降低了一点。第三：时间计算方面，笔者做啥事都习惯计算一下时间。所以当借到这只“母鸡”之后，定了一个月内必须要把母鸡还回去，这是最低的原则了。最终笔者在2月24号成功把这1万美刀（“母鸡”）转移到笔者的美股账户里面了（由于笔者在读书时代就经常去图书馆看一些自己感兴趣的书籍，其中就经常看到关于阿里巴巴的故事了，笔者也是个“问题宝宝”，所以也就“顺理成章”的了解到何为美国上市，何为纳斯达克，何为A股港股美股了）。经过分析，这也是笔者比较钟情于“美股”的原因。

资金准备好了之后，就是选股的问题了，笔者从读书时代除了喜欢看计算机的书籍，也喜欢看一些互联网公司的书籍。看着看着那一股脑就上来，好像总想干点啥的，而事实上到现在啥事也干不成，不过来日方长，只要还没意外死亡，也就还有点时间。所以以前看了不少这些公司的书籍，笔者又是问题宝宝的原因，所以就不断的把疑问弄明白了，最终也就可以尝试的去分析一家公司的商业模式，业务逻辑，以及未来的发展前景了。笔者是在2019年4月份注册的美股账户，不过也没有进行大额度的交易，小白入门，也就从“模拟炒股”开始了。经过差不多一年的沉淀积累，令我印象最深刻，股价又比较便宜的公司，也就只有“蔚来汽车”这家公司了。因为深入了解这家公司，所以最终也就购买这家公司的股价了，经常看到很多“金融佬”或“非金融佬”都常说不要把鸡蛋放在一个篮子里面，虽然笔者也是非常害怕这样的冒险，但也必须得尝试一下把鸡蛋放在一个篮子里面，经过分析，笔者也相信了自己的判断了，所以最终就把这1万美刀在蔚来汽车股价平均还是3.7到3.85美元一股的时候，以3.82的价格1万美刀全部买入了（一共是3.82乘2600股）。笔者把这所有的鸡蛋都放在一个篮子里面了，也始终相信蔚来汽车的发展前景，相信蔚来在未来不久会上涨，所以笔者最终是想在买入股票的15到20天之后卖出的，不论是盈利还是亏损，要把母鸡还回去。没想到这就刚好在买入股价的第二天也就是2月25号当天下午，就看到“蔚来中国总部将落地合肥，计划融资百亿”的相关新闻以及公告了。所以股价还没在晚上10点半开盘，在盘前交易（美股是有盘前盘后交易的）就已经上涨最高达30%了，也就是股价从不到4块钱涨到5块多一点点了，笔者因为有一点胆小的原因，并且也理性分析，所以这样的涨幅也是比较“妖”的感觉，大家都涌上来不一定是啥好事，笔者也在晚上开盘的时候以4.96的价格全部抛售了蔚来的所有股票了。也就是说在一夜之间瞬间盈利了将近3千美金。连笔者也想不到在短短的一夜之间就能盈利了这点小钱，毕竟在此之前从来没有尝试过超过2000块钱RMB的金融投资。不过也证明了笔者的分析判断能力，所以在抛售蔚来的第二天，股价也就慢慢往下跌了，过了两三天之后，股价就一直平均在4到4.5之间的幅度。直到今天3月7号早上看了一下，盘后交易股价已经跌倒3.5块钱了。不过如果是长期投资，我还是认为蔚来汽车会发展得更好的，毕竟创始人背景不像乐视的贾跃亭那样有点虚并且不靠谱。蔚来的创始人还是比较专心的做电动车，所以也就积累了一大批的“小迷弟”了，长期投资我还是非常看好蔚来汽车的，如果是短期的话，风险还是有点大。而笔者这次的“借鸡生蛋”做的就是短期投资，主要以回笼资金为主，不贪图赚大钱，所以见好就收了。

以上说了这么多，都只是在做空瑞幸咖啡股价的前面小插曲。说起瑞幸咖啡，还有一家公司甚是让笔者印象极度深刻。就是“拼多多”了，拼多多从创办到美股上市也不过两年多一点点左右，已经破了中国公司上市记录了，瑞幸也就更厉害了，才创办一年多就美股上市了。所以这两家公司从上市到现在，笔者偶尔有时间都会去关注一下股价以及其他方面的信息。笔者比较害怕风险，所以每次做啥事情之前都会收集大量的数据来分析一下，瑞幸咖啡在刚上市的时候，我就立下了“flag”，他日有机会的话，必然会买瑞幸咖啡的股价。笔者也有尝试过瑞幸咖啡的饮品，也跟星巴克比对了一下，这个模式也还是可行的，未来也有极大的可能性会占领更多的中国市场。只要瑞幸咖啡不是资本家们的“游戏”，否则将会损害更多投资者的利益。

所以在做了蔚来的投资之后，笔者就开始瞄准瑞幸咖啡了，但是不选择做多（有两个词语可能运维兄弟以及其他的IT兄弟不懂，所以简单的解释一下）。“做多”的意思是：买股票上涨；例如笔者在蔚来股价3.82的时候大量买入了，然后在4.96的时候卖出，赚取了中间的差价，这就是做多。“做空”的意思是：买股票下跌，例如笔者在瑞幸咖啡40.52卖出100股，然后瑞幸下跌到34元买入100股来平衡，赚取中间的差价，这就是做空，一旦瑞幸咖啡上涨超过了40.52，那么就亏损了。笔者并非专业的“金融佬”，所以只能这样描述了，百度还有更通俗以及专业的解释。

笔者做了一个小市场调查，发现疫情发展期间很多餐饮行业经济都遭到大量损失，而瑞幸咖啡刚好就是餐饮行业了，疫情初期股价还能在51美元左右，所以在2月初的时候，各大互联网以及财经频道，都传出了瑞幸咖啡被美国知名“做空机构（浑水公司）”爆出瑞幸咖啡的各种造假“传闻”，所以导致股价从50多一点跌倒40多一点点了。不过笔者不管是中国互联网传出的，还是YouTube传出的各种文章，视频也罢。笔者没有太多时间去研究分析这些东西（当然分析这些也有一定“优势”的）。笔者看到互联网很多都涌现到这份“做空报告”，而笔者就把眼光瞄准到中国“宏观”的世局去，只关注疫情发展事态和餐饮行业的点。所以估算瑞幸咖啡未来一段时间还会继续跌，因为有了蔚来的一次收益，所以笔者也有一点点小开心（本来想暂时退出股市把母鸡还回去的），但还是要理性分析思考，觉得这做空瑞幸应该是一个非常好的机会，要尝试一下，不可放弃，哪怕结果真的不大如意。

所以经过笔者用心做了一天的市场调查和数据分析，最终决定在2月26号开始进行做空瑞幸咖啡，所以当晚开盘的时候，笔者通过观察最终以40.52的价格卖空瑞幸咖啡，至于打算在瑞幸跌到多少再买回来，笔者不能确定。笔者只能确定瑞幸咖啡会跌，但是跌到多少，笔者也无法预计，毕竟也是个业余的，没有“金融佬”那般精准分析。所以当卖空交易成功的时候，笔者后续打算观察几天的情况，然后在第二天中午，笔者浏览了一下财经新闻，以及互联网的一些新闻，发现美股这周（2月24到28号的一周）的整体趋势是持续下跌。跌到自2008年金融危机以来最低的一周。也发现很多散户投资者都亏损了很多钱，然而笔者的眼光是放在做空，买瑞幸股价不断下跌，因为在做空交易次日关注到这些财经新闻，发现瑞幸已经从40元盘前慢慢下跌到“39元、38元、37元...”，笔者有打算在37元的时候买入瑞幸股价，这样就平衡了40.52卖出的股价，然后中间将会赚取3块多的利润。但由于是盘前交易，还没到晚上十点半的美股开盘，所以笔者就克制自己了，耐心等到开盘看一下。（在此笔者还有一个“小插曲”，在卖空瑞幸咖啡的时候，加了3.6倍杠杆，就是在做空之前，笔者的美股账户里面的金额将近1.3万美元，然后证券平台最低要求2千美金就可以支持做空股价。所以笔者的最高支持做空股价为3.6倍，通俗来讲就是1.3万乘3.6，所以笔者做空卖出瑞幸股价的价值达到了4万多美元了，不得不佩服这个“杠杆原理”了，用小的钱可以撬多的钱。）

笔者用了杠杆，手头上卖出了瑞幸咖啡的股价达到了1145股，次日27号晚上开盘的时候，笔者就一直盯着瑞幸咖啡的股价数字，就看看这货能跌到多低，我也是极度“紧绷”的（“金融佬”长期处于极度紧绷的状态，所以大多数金融佬都有非常“变态”的一面。在这不多说了）。最终熬了一个多小时之后，发现瑞幸居然跌到了“35元、34元”了。笔者已经深深的感到“警告”的声音了，最终笔者在以股价跌到34.88进行平衡仓库买入了1145股，也就是说笔者在这中间盈利了6块多美元每股。综合收益是：做空瑞幸咖啡盈利了6千多美金，做多蔚来盈利了将近3千美金。所以笔者总收益是1万多一点点美金。笔者也是远远没有想到短短几天就盈利了这点金额，所以在后续又继续做多一次蔚来了，不过看着蔚来这样的“情况”，似乎不妙；又看到全球的疫情局势，未免有点担心了，最终抛售了蔚来走人，活命要紧，以及需要把母鸡还回去那几个平台了。没想到第二次做多蔚来之后，第二天这个全球股市跌得更厉害了，简直“黑色星期五”。A股港股美股的所有指数都在下跌，很多朋友的基金，理财也遭到一点损失了。所以笔者不得不庆幸自己没有“太贪”，避免了收益的金额再遭受到“灭顶之灾”（笔者的收益金额，由于在第二次做多蔚来的时候进行了及时止损，所以没有遭受到损失），笔者也看到有其他做空瑞幸的散户投资者一个晚上盈利了20多万美金。笔者也比较奉行一句“君子爱财，取之有道”。最终笔者在昨天也就是3月6号提现全部的资金出来，把“母鸡”送回去那几个平台了，而自己则留下了鸡蛋。

或许有人看到了说“你这么有时间去研究股票，工作上会不会就没有那么上心了？会不会经常摸鱼了？”笔者想说的是：这是美股，中国时间晚上才开盘的（A股港股就是白天）。每个人都是同样的24小时，有的人可以每天都有一点时间去玩游戏，追剧，看各种抖音快手直播。但是也有很多人每天可以花一点时间去背英语单词，考一些国内甚至国际的证书。我看到很多人在非工作期间也可以进行做一些自媒体、写文章、做投资、开直播去赚钱的。所以笔者恰好对玩游戏、看抖音快手这些完全没有兴趣，自然也就“有时间”来研究一下投资，以及做粤菜了。笔者这次的投资也是在疫情时期，所以时间就稍微充足一点了。笔者也会在三月份开始慢慢进行投简历找工作了，毕竟最直接的收入还是认真工作。其次才是投资或别的收入。

上面说了这么多，笔者的意思是想让我们运维的兄弟，IT的兄弟可以培养一些“财商”，也就是投资和理财。咱们大多数的IT行业者薪资普遍比一般岗位高那么一点点，如果能好好利用自己的小钱来做“钱滚钱”，做一些理财、基金、股票投资作为“睡后收入”也是蛮不错的。不至于等哪天裁员不幸轮到自己了，这压力就一下子上来了。有能力有口才的可用写文章，做直播讲师等等一些擅长的副业，反之研究股票投资是最好的“副业”。

笔者也提醒一下广大的运维、IT朋友，没事的时候也最好训练一下自己的阅读习惯，笔者还在读书的时候就经常每周末跑去图书馆阅读书籍了，阅读多一点不同类别的书籍，对大脑逻辑思考方式也会有所不同，不至于思维永远定格在这里，或许很多人都不爱看书，但是笔者建议可以先看自己非常感兴趣的相关内容书籍开始，这样就可以慢慢的培养这个习惯了。例如笔者就是在以前读书的时候看的计算机书籍、Linux书籍、然后再到一些商业书籍、投资理财书籍、风险控制、再到现在一些法务的书籍等等。然后这个习惯就出来了，也就适应了，头脑思考也变得独立镇定，不至于遇到啥事都那么慌张了。很多时候遇到一些问题，在这个真真假假的互联网世界里面是非常难分辨的，各种坊间传言太多了，所以看多一点不同的书籍，这样自己的判断能力也会有所提升。2月底“风靡一时”的“微盟删库事件”，影响非常大了，我们运维也应该有所意识，倘若看多一点不同行业的书籍，例如法律相关的，那么估计就不会直接删人家的数据库了。释放了个人的愤怒，但是也损害了企业和个人的职业前途。这是多么亏损的“投资”了。在这个时代，要报复一个人的方案有很多种，就法律范围内的“正当报复”，以及法律范围内的“非正当报复”，这样都可以解决个人的“超级愤怒”。不过笔者也不提倡这些方法，归根结底还是自己“情绪管理”的问题，所以笔者也建议咱们IT的兄弟也要控制一下自己的情绪，对待事物要抱有“敬畏之心”，看多点书籍就可以了！笔者在做空瑞幸的时候，情绪也是比较激烈的，毕竟是第一次“玩做空”。但是也努力控制自己了，所以这也是笔者的一次“情绪控制突破”吧。没有大喜大落。

咱们IT程序员也要拓展一些别的业余兴趣爱好。不至于除了在黑屏绿字里面，外面啥都没有了。工作只是自己的一部分，拓展业余爱好也是比较重要，笔者的业余就是投资股票和粤菜了。说到业余爱好，笔者也举两个例子。第一：笔者有个朋友是做餐饮行业的厨师，但是他的业余爱好居然是计算机，笔者这个朋友在有空的时候都会去钻研计算机的技术，所以笔者有时候跟这位朋友进行计算机讨教的时候也是非常愉悦。第二：笔者有个熟人是初中毕业的，工作一直在小工厂的流水线里面，但是这个熟人心思还算单纯，没有太多别的想法了，所以业余爱好就是玩钢琴，很久以前是从电脑的钢琴软件开始玩，然后电脑键盘弹的钢琴非常熟练，后来他自己存了一点钱，就买了一架电子琴，弹得也就更加熟练了，或许将来会有“高手”发现这块“璞玉”，进行雕琢。所以工作只是生活的一部分！

笔者也建议我们运维兄弟做一些“物资储备”。以前看书总是看到一句话“兵马未动，粮草先行”，只要随时随地的做好准备，就可以在“雷天暴雨”来的时候可以进行“对抗”了。不至于死的太惨。我们不能心存侥幸，觉得事情都不会发生在自己身上，都是在别人身上发生，殊不知或许下一个自己就沦为“别人”了。所以要做到“居安思危，未雨绸缪”，只有手中存储了大量的“物资”，才能不慌不忙！

笔者以前在书本看到一条甚好的理财法则“4321理财法则”，就是说自己家庭的资产配置为40%用来做投资增值，30%做生活开销，20%做储蓄备用，10%做购买一些家庭必备的商业保险。只有这样安排整个家庭的现金流才会比较健康。可能有很多人觉得这个实际上应该如何操作，完全不懂咋办？这点笔者认为没有不懂的东西，只有不学的知识。咱们业余的，不需要学得跟“金融佬”那般“高深莫测”，懂的“基本操作”也足够了。至少比啥都不懂好了很多，倘若有个别的兄弟们对数字，分析，逻辑颇有天赋，那么钻研一下股票投资就更事半功倍了，甚至将来有可能堪比“金融佬”。笔者天资愚钝，脑子没有很多人的好使，有太多的“为什么”了，对于投资角度，只能从“宏观”分析，还暂时无法做到“微观”分析，所以笔者对股票的K线图，财务报表，等等一些也是看不懂的。

笔者已经写了不少，再说一下这个“4321理财法则”的配置吧，笔者在家里是组织全家人的资金进行套用这个理财法则的，比如说40%的家庭财产是用在“长期持有”股票投资，30%是用来家庭生活支出，20%用来做支付宝的基金，定投理财（支付宝也是一个相当专业的金融工具，可以多去了解挖掘一下，不要把钱存在银行，没有啥利息，反而会扣某些费用，存在支付宝里面的低风险定期理财更好），10%用来配置家庭的保险，给爸妈买一套大病保险，给自己买一套大病保险和意外身亡保险（倘若有天意外不测，还有一百万保险金给家人。对于商业保险，欧美地区是比较注重的，但国内还有很多人甚至都没有了解，自然就产生了“轻松筹”、“水滴筹”的一些机构了），再给外甥女买一份教育基金（每个月存一点钱到这个教育基金，然后小孩读大学的时候就有一笔钱的了）。说到教育基金，建议广大的运维兄弟可以去支付宝或者一些保险公司了解一下这个，或许对已经有小朋友或者打算要小朋友的还是有极大的帮助的。毕竟也是提前为小朋友准备一笔教育资金储备了。

笔者再说一下股票投资的“策略”。首先笔者认为长期持有股票的公司分别有“小米集团”，“拼多多”，“达内教育”，“蔚来汽车”，“瑞幸咖啡”，东南亚巨头公司“SEA”。

第一：小米集团的背景不多说了，只要创始人没出什么幺蛾子，还是可以值得长久持有的，毕竟小米集团能从非世界500强公司进入到500强了。

第二：拼多多，这家能在京东淘宝电商独步天下的时候，还能在电商抢到一杯羹，以及在短短的两年就上市了，去年市值一路追赶超过京东，所以拼多多合适做多和做空。

第三：达内集团，相信很多中国的运维、IT从业者都听过这家“IT培训机构”了，那个名声也是IT培训业界相当差的一家公司了，差的出名过火。但是笔者偏偏非常看好这家公司，这家公司“口碑差”，可能微观上高层管理不善导致的（没有深入了解），但是笔者看好它的未来，只要创始人在公司倒闭之前想好对策，长久持有这公司股价也是相当不错。就去年笔者陪同一个朋友在深圳找游戏插画培训机构，笔者到每家机构的时候都会“顺便”咨询一下“达内集团”，共陪朋友找了5家游戏插画机构比对，每家都说达内不好，在培训行业内相当差。笔者是在2017年下半年无意中了解到达内集团的，当时在互联网无意中看到几集达内北京总部老师的公开课视频，发现有两个Linux老师讲课相当风趣幽默（分别是NB老师，DMY老师了）。笔者当时一下子就成了这两个老师的“小迷弟”，那个时候也偶尔关注了“达内集团”，所以让笔者相信达内的发展只是因为这两个“风趣的老师”了，非常期待以后有机会能够认识这两位老师。就在去年达内股价跌到不到1块美金，以及到1块多的时候，笔者的模拟账户大量的购买了这家公司的股价，今年1月份平均涨到了2块多一股，后来笔者还是持续看好，结果在2月下旬的时候，达内更换了高层CFO，更换了审计事务所。股价一下子涨了翻倍。笔者模拟炒股账户里面是在去年9月份以1块多买入，然后股价涨到了4块多的时候全部抛售了，综合模拟账户收益达到了140%的回报。未来打算也会用“真金白银”去长期持有。毕竟疫情期间或者疫情过后在线教育也将会是一个未来的趋势（以前学习需要去课室，前两年学习需要电脑，现在学习在TV、PC、移动端都无缝支持），达内刚好也是在线模式了。

第四：蔚来汽车，这家公司的背景也是比较大，有腾讯投资（腾讯的眼光也是挺独特的，腾讯的法务团队在深圳南山也被行内称为“南山必胜客”，可以多去了解一下），有合肥政府投资，创始人李斌跟笔者榜样也是北大毕业的。（在此说一下笔者的榜样：笔者的榜样是新东方集团的创始人“俞敏洪，俞老师”，笔者倒觉得俞敏洪跟蔚来的李斌有一些相同之处。第一：都不玩虚。第二：都是农村出来的。第三：都是北大毕业的。第四：都有不怕死的精神。第五：都是做熟的东西，俞敏洪一直做教育，李斌一直做车。）蔚来做的是新能源电动车，虽然现在还是特斯拉做的相当好，但是总有一天也相信咱们国产“蔚来”也可以“逆袭”。笔者除了看好蔚来的发展前景，也非常看好创始人。所以蔚来汽车也合适长久持有。（做虚的企业就有乐视的贾跃亭了，啥都想干，结果一不小心一家巨头上市公司就干没了，所以小心谨慎，以及做某件事专注一点也是相当重要的）

第五：瑞幸咖啡，这家公司比拼多多更厉害，成立了一年多就上市了。创始人背景也是相当厉害，没有OFO的创始人那样“初生不怕牛犊”，都是经验相当“老辣”的了，但是这家公司的股票风险极高。企业发展太过于迅速也不一定是“好事”。如果不是幕后资本家们的“资本游戏”，那么瑞幸咖啡还是非常合适长久持有的，反之就相当合适“做空”了。笔者也是还没完全弄明白这货。如果长久持有的话，笔者不会放太多美金到这货。

第六：东南亚巨头“SEA”，这家公司有个小称呼为“中国的阿里腾讯”的小合体。这家公司做的是电商板块和游戏板块，是新加坡的企业。东南亚的互联网体系现在还没有完全跟上中国，所以还是有很大的发展前景，这货非常合适长久持有，笔者也大致看了一下这公司的股价，发现从上市到现在持续上升状态，根据网友的财报分析也说明这家公司一直属于盈利状态。不像蔚来，瑞幸一直处于烧钱状态。笔者认为股票投资最好的“策略”就是买一些自己信得过不玩虚的公司，然后长期的持有陪伴该公司成长，最终公司发展的越好，自己手头上的股票就越值钱了。太过于急功利，反而容易遭受损失。

最后：想投资的话，笔者认为要先从模拟炒股开始练习，其次再到支付宝的基金理财定投这些。支付宝的东西风险还是属于中低的，普通人都可以接受。如果想买美股港股的话，笔者建议要办理一张香港银行卡。这样境内境外资金操作就方便一点。如果不懂得如何办理或者有啥限制，可以百度搜一下具体流程，也可以问一下笔者，笔者非常乐意给运维的兄弟们解答。
